# 📊 Tableau Assignment 6 – Parameters & Groups

## 📌 Project Overview
This assignment demonstrates the implementation of **Parameters** and **Groups** in Tableau to create dynamic and interactive visualizations.

## 🎯 Objectives
- Create a **View By** parameter with the following values:
  - Region
  - Category
  - Sub-Category
- Display Sales based on the selected parameter.
- Update the worksheet title dynamically according to the selected parameter.
- Create product groups for:
  - Acco
  - 3M
  - Samsung
  - Apple
  - Xerox
  - Others
- Display the **Average Discount (%)** for each product group.

## 🛠️ Tools Used
- Tableau Public
- Sample Superstore Dataset

## 📈 Features
- Parameter Creation
- Dynamic Sales Analysis
- Dynamic Worksheet Title
- Product Grouping
- Average Discount Analysis

## 📚 Key Learnings
- Parameters
- Calculated Fields
- Groups
- Dynamic Visualizations
- Interactive Analysis
