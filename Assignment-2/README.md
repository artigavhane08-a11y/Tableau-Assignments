# Assignment 02 – Netflix Titles Analysis using Inner Join

## Objective
Merge Netflix Titles dataset tables using Inner Join and identify the actor/actress with the highest number of appearances in the **Sports Movies** category. Display the actor/actress name along with the corresponding movie titles.

## Dataset
- Netflix Titles Dataset

## Tools Used
- Tableau Public

## Concepts Covered
- Inner Join
- Data Combining
- Data Filtering
- Distinct Count
- Color Highlighting

## Files Included
- Assignment-02.twbx

## Output
This visualization identifies the actor/actress with the highest number of appearances in Sports Movies and displays the corresponding movie titles using an interactive Tableau visualization.
