
# Assignment 03 - LODs & Histogram

## Objective
Create a Tableau histogram displaying the count of customers based on their order frequency, with the number of orders on the X-axis and the corresponding customer count on the Y-axis.

## Dataset
- Sample Superstore Dataset

## Tools Used
- Tableau Public

## Concepts Used
- Level of Detail (LOD) Expression
- Histogram
- Calculated Field
- Count Distinct
- Data Visualization

## Output
A histogram showing the distribution of customers based on the number of orders they placed.
