# 📊 Tableau Assignments

## 📌 Repository Overview

This repository contains my Tableau practice assignments completed using the **Sample Superstore** and **Netflix Titles** datasets. These assignments cover fundamental and intermediate Tableau concepts, including tables, joins, Level of Detail (LOD) expressions, sets, KPI cards, parameters, and groups. Each assignment focuses on solving a real-world business problem while improving data visualization and analytical skills.

---

## 🛠️ Tools Used

- Tableau Public
- Sample Superstore Dataset
- Netflix Titles Dataset

---

## 📂 Assignments

### 📋 Assignment 01 – Table

**Objective**
Create a table showing **Category, Sub-Category, Region, Sales, and Profit** with **Grand Total** and **Percent of Total**.

**Concepts Used**
- Text Table
- Grand Totals
- Percent of Total
- Table Calculations
- Formatting

---

### 🎬 Assignment 02 – Netflix Titles Analysis using Inner Join

**Objective**
Merge Netflix Titles dataset tables using an **Inner Join** and identify the actor/actress with the highest number of appearances in the **Sports Movies** category. Display the actor/actress name along with the corresponding movie titles.

**Concepts Used**
- Inner Join
- Data Combining
- Data Filtering
- Distinct Count
- Color Highlighting

---

### 📊 Assignment 03 – LODs & Histogram

**Objective**
Create a histogram displaying the count of customers based on their order frequency, with the number of orders on the X-axis and the corresponding customer count on the Y-axis.

**Concepts Used**
- Level of Detail (LOD) Expressions
- Histogram
- Calculated Fields
- COUNTD (Count Distinct)
- Data Visualization

---

### 🥧 Assignment 04 – Sets

**Objective**
Create a pie chart comparing the **Top 50 Customers** and **Other Customers** based on total Sales using Tableau Sets.

**Concepts Used**
- Sets
- Pie Chart
- Sales Analysis
- Labels & Percentage
- Data Visualization

---

### 📈 Assignment 05 – Profit KPI Dashboard

**Objective**
Create a Profit KPI Card to display the latest year's profit, compare it with the previous year, and visualize the monthly profit trend.

**Concepts Used**
- KPI Card
- Table Calculations
- Year-over-Year (YoY) Analysis
- Conditional Formatting
- Line Chart
- Dynamic Growth Indicators

---

### ⚙️ Assignment 06 – Parameters & Groups

**Objective**
Create dynamic visualizations using **Parameters** and **Groups** to analyze Sales and Average Discount.

**Concepts Used**
- Parameters
- Groups
- Calculated Fields
- Dynamic Worksheet Titles
- Interactive Analysis
