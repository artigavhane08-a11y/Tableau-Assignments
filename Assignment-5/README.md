# 📊 Tableau Assignment 5 – Profit KPI Dashboard

## 📌 Project Overview
This Tableau dashboard presents a Profit KPI Card for the latest year in the dataset. It highlights the total profit, year-over-year (YoY) percentage growth, and a monthly profit trend using a line chart.

## 🎯 Objectives
- Display total profit for the current year (latest year in the dataset)
- Compare profit with the previous year
- Show percentage growth with a green upward arrow for positive growth and a red downward arrow for negative growth
- Visualize monthly profit trends using a line chart

## 🛠️ Tools Used
- Tableau Public
- Sample Superstore Dataset

## 📈 Dashboard Features
- 💰 Total Profit KPI Card
- 📊 Year-over-Year Profit Growth (%)
- 🟢/🔴 Growth Indicator (Arrow)
- 📉 Monthly Profit Trend Line
